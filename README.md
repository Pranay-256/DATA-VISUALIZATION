# DATA-VISUALIZATION

This repository includes the concepts and projects made for data visualization using :

    1. Matplotlib
    2. Seaborn
And it clearly showcases the different types of plots in data visualization like:

    1. Relational Plots
    2. Categorical Plots
    3. Distribution Plots
    4. Regression & mixed plots
    5. Matrix/Grid Plots
And at Last an IPL 2022 Analysis and Visualization project
